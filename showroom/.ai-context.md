# Fantasy Bakery Showroom — AI Context

## What This Project Is
A **Vite + React** 3D product showroom for Fantasy Bakery (Mombasa, Kenya).
It renders bakery products (bread, sugar rolls, buns) as interactive 3D GLB models
with a plastic-wrap overlay effect, loaded from `/public/models/`.

## Stack
| Layer | Tech |
|-------|------|
| 3D Engine | React Three Fiber (R3F) + Drei |
| State Bridge | Zustand |
| Shader | GLSL via Drei `shaderMaterial` |
| Bundler | Vite |
| Dev port | `http://localhost:5173` |
| GLB source | Meshy AI exports → `public/models/*.glb` |

## Agent Boundary (CRITICAL)
- **Antigravity** = Engine. Owns `<Canvas />` and everything inside it.
- **Cursor** = UI. Owns `ProductSelector.jsx` and the panel outside the canvas.
- **Bridge** = `src/store/useShowroomStore.js` — the ONLY place cross-agent state lives.
